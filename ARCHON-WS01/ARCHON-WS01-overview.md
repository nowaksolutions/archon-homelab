## Overview

**ARCHON-WS01** is a dedicated digital forensics and security analysis workstation designed to support investigative workflows, artifact analysis, and controlled execution of suspicious files.

While capable of functioning as a standard Windows endpoint, its primary role is to operate as a **hands-on DFIR platform** for real-world analysis and skill development.

---

## Primary Functions

**Digital Forensics Analysis** - Examination of file systems, logs, and artifacts to identify indicators of compromise and reconstruct activity  

**Malware Observation (Controlled Execution)** - Safe behavioral observation of executables within a controlled environment  

**Artifact Triage & Investigation** - Processing suspicious files, reviewing metadata, and validating potential threats  

**Log & Event Analysis** - Correlating endpoint activity with centralized logging for detection validation  

---

## Core Tooling Stack

**Disk & Artifact Analysis**  
Autopsy - File system analysis and timeline generation  
FTK Imager - Disk imaging and evidence preview  

**Memory Analysis**  
Volatility - Memory forensics and process inspection  

**Log & Event Analysis**  
Splunk - Log aggregation and correlation  
Windows Event Viewer - Native log analysis  

**System Inspection**  
Sysinternals Suite - Process, persistence, and system visibility  

**Network Analysis**  
Wireshark - Packet capture and protocol inspection  

**Malware Analysis**  
Flare VM - Controlled malware analysis environment
